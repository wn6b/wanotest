## نبذة عن المشروع
**Projects Bots** هي منصة برمجية متطورة مصممة لتكون الوجهة الأولى لإدارة ونشر مشاريع البوتات والتطبيقات. تم بناء المنصة لتعمل بالكامل بتقنيات الذكاء الاصطناعي، حيث توفر تجربة مستخدم فريدة تجمع بين السرعة، الأمان، والتصميم المستقبلي (بالاعتماد كلياً على أيقونات الفيكتور الاحترافية وتجاوز الإيموجيات التقليدية) بما يليق بمعايير عام 2026.

---

## المميزات الرئيسية (Core Features)

### 1. الرقابة الذكية للحسابات (AI Profile Monitor)
* **فحص حي للبيانات:** يقوم الذكاء الاصطناعي بفحص (اسم المستخدم، البايو، ورابط الصورة الشخصية) قبل حفظها في الإعدادات.
* **حظر تلقائي:** منع أي محتوى غير لائق أو ألفاظ بذيئة من الحفظ في قاعدة البيانات فوراً للحفاظ على بيئة المنصة.

### 2. نظام AI Scanner الأمني للملفات
* **فحص ذكي:** فحص تلقائي لكل مشروع يتم رفعه للتأكد من خلوه من الفيروسات أو التهديدات البرمجية.
* **تحليل Claude AI:** الربط مع موديل `claude-sonnet-4-20250514` لتحليل وصف المشروع ومحتواه برمجياً وإصدار قرار النشر.

### 3. لوحات تحكم متطورة (Dashboards)
* **لوحة تحكم المالك (Owner):** تحكم كامل في المنصة، تخصيص احترافي للحساب، إدارة المشاريع والمطورين، سجل النشاطات، وإحصائيات حية.
* **لوحة تحكم المستخدم (User):** واجهة مخصصة لتصفح المشاريع المتاحة، الوصول السريع لروابط المطور، والتفاعل المباشر مع المنصة.

### 4. واجهة مستخدم (UI/UX) مستقبلية
* **اعتماد كلي على الـ Vector:** استخدام مكتبة FontAwesome لضمان تصميم نقي واحترافي.
* **شاشة تحميل ولوقو متطور:** رسومات SVG متحركة تعكس هوية المنصة التقنية وتوفر تجربة بصرية مذهلة.
* **تصميم 2026:** استخدام تدرجات لونية، تأثيرات زجاجية (Glassmorphism)، وتصميم متجاوب بالكامل مع كافة الأجهزة.

---

## التقنيات المستخدمة (Tech Stack)
* **Frontend:** HTML5, CSS3, FontAwesome (Vector Icons).
* **Logic:** Pure JavaScript (ES6+) - لا حاجة لأي إطارات عمل معقدة.
* **AI Integration:** Anthropic API (Claude AI).
* **Database:** LocalStorage API (لضمان السرعة الفائقة والعمل المباشر).
* **Fonts:** Syne & Space Mono (للطابع التقني) + Tajawal (للعربية الاحترافية).

## طريقة التشغيل والنشر (Deployment)
المشروع جاهز للعمل مباشرة كونه مبنياً بتقنيات الـ Front-end المستقلة، ولا يحتاج لبيئات تشغيل معقدة:
1. قم برفع ملفات المشروع (`index.html`, `style.css`, `script.js`) إلى منصة استضافة تدعم المواقع الثابتة مثل **GitHub Pages**.
2. بمجرد رفع الملفات وفتح الرابط، سيعمل النظام بالكامل وبشكل مباشر معتمداً على متصفح المستخدم والـ API.
